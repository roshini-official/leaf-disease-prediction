# 🌿 Leaf Disease Prediction System

## 📌 Overview
This project is a Machine Learning-based system that predicts plant leaf diseases from images. It helps in identifying whether a leaf is healthy or affected by a disease, enabling early detection and better agricultural decisions.

---

## 🎯 Objective
- Detect diseases in plant leaves using image data  
- Classify leaves into healthy or diseased categories  
- Build a simple and efficient prediction model  

---

## 🛠️ Technologies Used
- Python  
- NumPy  
- Pandas  
- Matplotlib  
- OpenCV (for image processing)  
- Machine Learning / Deep Learning model  

---

## ⚙️ Working Process
1. **Data Collection**  
   - Used labeled dataset of plant leaf images  

2. **Data Preprocessing**  
   - Resized images  
   - Normalized pixel values  
   - Converted images into suitable format for model  

3. **Model Training**  
   - Trained model to classify leaf conditions  
   - Learned patterns from image features  

4. **Prediction**  
   - Input: Leaf image  
   - Output: Predicted disease / healthy leaf  

---

## 🚀 Features
- Image-based disease detection  
- Simple and efficient workflow  
- Helps in early identification of plant diseases
